# PerfectBoard

보편적인 형태의 게시판 목록과 페이지네이션, 댓글, 대댓글 등을 구현해보는 프로젝트

### Project Stacks

---

#### Client

- React v17
- React Router v5
- TypeScript
- Axios
- Scss


#### Server

- Node.js v14
- Sequelize(ORM)
- MySQL
